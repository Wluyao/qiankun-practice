## 主应用

### main-app

进入main-app目录

#### 安装依赖

```
yarn
```

或者

```
npm i
```

#### 启动

```
yarn serve
```

或者

```
npm run serve
```

通过 [http://localhost:8080/](http://localhost:8080/) 访问主应用。
